# Пространственное неравенство доступности городской среды для маломобильных граждан в Москве

## Синопсис

Исследование посвящено анализу доступности городской инфраструктуры для маломобильных граждан в Москве на основе открытых геоданных. Работа фокусируется на пространственном распределении доступных объектов, сравнении административных округов и выявлении территориальных диспропорций.

Ключевая цель — определить, насколько равномерно развита доступная среда, и выявить скрытые различия при переходе от абсолютных показателей к нормализованным (с учетом численности населения).

## Актуальность

Доступная городская среда является важнейшим фактором качества жизни и социальной интеграции маломобильных граждан. Несмотря на развитие городской инфраструктуры, вопрос равномерности распределения доступных объектов остается открытым.

На практике доступность может значительно различаться между районами города: одни территории обладают развитой инфраструктурой, тогда как другие остаются слабо адаптированными.

В результате возможность передвижения и использования городской среды зависит не только от физических ограничений человека, но и от его географического положения внутри города.

---

### Процесс сбора и очистки

Данные OpenStreetMap были получены с использованием Overpass API. Первичный датасет содержал большое количество нерелевантных признаков.

В процессе обработки:

* отобраны ключевые признаки (координаты, доступность)
* значения wheelchair преобразованы в бинарный формат (1 — доступно, 0 — нет)
* удалены пропуски и шумовые данные
* выполнена агрегация по административным округам
* добавлены данные о численности населения

Обработка выполнялась с использованием Python (pandas).

## Анализ

### 1. Сравнительный анализ

Сравнение округов проводилось по следующим показателям:

* количество доступных объектов
* общее количество объектов
* доля доступных объектов
* индекс доступности (с учетом населения)

Результаты показали значительные различия между округами.

---

### 2. Анализ неравномерности

Рейтинг округов выявил выраженное территориальное неравенство.

Разница между лидерами и аутсайдерами указывает на неоднородное развитие городской инфраструктуры.

---

### 3. Анализ взаимосвязей

С помощью диаграммы рассеяния была исследована связь между численностью населения и уровнем доступности.

Результаты показали отсутствие прямой зависимости, что свидетельствует о несистемном развитии доступной среды.

---

### 4. Визуализация

В рамках исследования были построены:

* столбчатая диаграмма (индекс доступности по округам)
* круговая диаграмма (доля доступных объектов)
* диаграмма рассеяния (население и доступность)

---

### 5. Картографический анализ

Была создана интерактивная карта доступности объектов с использованием Google My Maps.

Карта позволяет:

* визуально оценить распределение доступных и недоступных объектов
* выявить зоны концентрации
* определить районы с низкой доступностью
